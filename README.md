# Personal Portfolio Website

This is a modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. It's designed to showcase your professional experience, skills, and contact information.

## Features

- Responsive design that works on all devices
- Modern and clean UI
- Smooth scrolling navigation
- Sections for About, Experience, Skills, and Contact
- Profile picture display
- Social media links
- Easy to customize and maintain

## Getting Started

1. Clone this repository to your local machine
2. Customize the content in `index.html` with your personal information
3. Replace the profile picture (`_DSF0143.JPEG`) with your own image
4. Update the social media links in the contact section
5. Customize the colors and styling in `styles.css` if desired

## Deploying to GitHub Pages

1. Create a new repository on GitHub
2. Push your code to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/your-repo-name.git
   git push -u origin main
   ```
3. Go to your repository settings on GitHub
4. Scroll down to the "GitHub Pages" section
5. Select the main branch as the source
6. Your site will be published at `https://yourusername.github.io/your-repo-name`

## Customization

- Update the content in `index.html` to match your experience and skills
- Modify the colors in `styles.css` to match your personal brand
- Add or remove sections as needed
- Update the profile picture and social media links
